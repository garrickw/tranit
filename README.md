This a small tool for translating. Base on the command line and youdao API.
Word or sentence are both excepted.
English or Chinese are both excepted.

usage example:

./tranit.py cool

result===>

#很酷的
#--------------------
#n. 凉爽；凉爽的空气
#adj. 凉爽的；冷静的；出色的
#vt. 使…冷却；使…平静下来
#vi. 变凉；平息
#adv. 冷静地
#n. (Cool)人名；(法)科尔；(英)库尔
#--------------------
#From web:
#cool:
#凉爽的; 酷; 酷
#Cool jazz:
#冷爵士乐; 冷爵士; 冷爵士乐
#3Tumble Cool:
#用烘干机低温烘干; 打冷风; 用烘焙机低温烘焙

./tranit.py 天气
